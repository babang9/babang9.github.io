Water9 개인정보 처리방침

최종 업데이트: 2025년 11월 11일

핵심 원칙: Water9는 어떠한 개인정보도 수집, 저장, 전송하지 않습니다. 모든 데이터는 사용자의 기기에만 저장됩니다.

1. 개요

Water9("앱")는 사용자의 물 섭취량을 추적하는 완전한 오프라인 애플리케이션입니다. 본 개인정보 처리방침은 앱이 사용자 데이터를 어떻게 처리하는지 설명합니다.

2. 수집하는 정보

Water9는 어떠한 개인정보도 수집하지 않습니다.

앱은 다음과 같은 정보를 수집, 전송 또는 공유하지 않습니다:

  • 이름, 이메일 주소, 전화번호 등의 개인 식별 정보
  • 위치 정보 (GPS, IP 주소 등)
  • 기기 식별자 (IMEI, MAC 주소 등)
  • 사용 패턴 또는 분석 데이터
  • 연락처, 사진, 파일 등의 기기 데이터
  • 건강 데이터 (물 섭취 기록은 로컬에만 저장)

3. 로컬 데이터 저장

앱은 다음 데이터를 사용자의 기기에만 저장합니다:

  • 물 섭취 기록: 날짜, 시간, 섭취량
  • 사용자 설정: 일일 목표량, 컵 프리셋, 알림 설정
  • 테마 설정: 다크 모드 선호도

이 데이터는 앱의 기능 제공을 위해서만 사용되며, 절대로 외부로 전송되지 않습니다.

4. 데이터 공유

Water9는 어떠한 데이터도 제3자와 공유하지 않습니다.

앱은 완전한 오프라인 모드로 작동하며:

  • 인터넷 연결이 필요하지 않습니다
  • 외부 서버와 통신하지 않습니다
  • 분석 서비스를 사용하지 않습니다
  • 광고 네트워크가 없습니다
  • 소셜 미디어 통합이 없습니다

5. 권한

Water9가 요청하는 권한:

5.1 알림 권한

  • 목적: 물 섭취 알림을 보내기 위함
  • 사용: 사용자가 설정한 시간에 로컬 알림 표시
  • 선택 사항: 사용자가 거부할 수 있으며, 앱의 핵심 기능은 계속 사용 가능

요청하지 않는 권한:

  • 위치
  • 카메라
  • 연락처
  • 사진 라이브러리
  • 마이크
  • 기타 개인정보 관련 권한

6. 데이터 보안

모든 데이터는 사용자의 기기에 저장되므로, iOS의 기본 보안 메커니즘(기기 암호화, 앱 샌드박스)에 의해 보호됩니다.

앱은 추가적인 암호화를 사용하지 않으며, 데이터는 SQLite 데이터베이스와 SharedPreferences에 저장됩니다.

7. 아동 개인정보 보호

Water9는 4세 이상의 모든 연령대가 안전하게 사용할 수 있는 앱입니다. 앱은 어떠한 개인정보도 수집하지 않으므로, 아동의 개인정보 보호에 대한 우려가 없습니다.

8. 데이터 백업 및 복원

앱 데이터는 iOS의 자동 백업 시스템을 통해 iCloud 또는 iTunes에 백업될 수 있습니다. 이는 사용자의 기기 설정에 따라 결정되며, 앱은 백업 프로세스를 제어하지 않습니다.

중요: 앱을 삭제하면 모든 데이터가 영구적으로 삭제됩니다. 백업이 없으면 복구할 수 없습니다.

9. 제3자 서비스

Water9는 어떠한 제3자 서비스도 사용하지 않습니다.

앱은 다음을 사용하지 않습니다:

  • 분석 서비스 (Google Analytics, Firebase 등)
  • 광고 네트워크
  • 크래시 리포팅 서비스
  • 소셜 미디어 SDK
  • 결제 처리 서비스

10. 사용자 권리

앱이 개인정보를 수집하지 않으므로, 다음과 같은 권리를 행사할 필요가 없습니다:

  • 데이터 접근 요청 (모든 데이터는 이미 사용자 기기에 있음)
  • 데이터 삭제 요청 (앱을 삭제하면 모든 데이터 삭제됨)
  • 데이터 이동성 (수집된 데이터가 없음)

사용자는 언제든지 앱을 삭제하여 모든 로컬 데이터를 제거할 수 있습니다.

11. 쿠키 및 추적 기술

Water9는 쿠키, 웹 비콘, 추적 픽셀 또는 기타 추적 기술을 사용하지 않습니다.

본 개인정보 처리방침은 필요에 따라 업데이트될 수 있습니다. 중요한 변경사항이 있을 경우, 앱 업데이트를 통해 사용자에게 알립니다.
