개인정보처리방침

시행일: 2026년 6월 28일  |  최종 수정: 2026년 6월 28일

물때 앱은 사용자의 개인정보를 수집하거나 외부 서버에 전송하지 않습니다.
위치 정보는 기기 내에서만 처리되며, 어떠한 형태로도 저장·공유되지 않습니다.

1. 수집하는 정보

물때 앱은 아래와 같은 정보를 처리합니다.

  • 위치 정보 (선택): 사용자가 "내 주변 관측소 찾기" 기능을 직접 실행할 때 한시적으로 사용됩니다. 가까운 조위관측소를 찾는 목적으로만 사용되며, 저장되거나 외부로 전송되지 않습니다.
  • 즐겨찾기·최근 관측소: 사용자가 선택한 관측소 정보(이름, 코드)가 기기 내 로컬 저장소에만 저장됩니다. 서버 전송 없음.

2. 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 식별 가능한 개인정보
  • 기기 식별자(IDFA, GAID 등)
  • 앱 사용 행동 데이터·분석 데이터
  • 결제 정보

3. 위치 권한 안내

앱은 Android의 ACCESS_FINE_LOCATION 및 ACCESS_COARSE_LOCATION 권한을 사용합니다.

  • 권한은 사용자가 "내 주변 관측소" 버튼을 누를 때에만 요청됩니다.
  • 위치 정보는 가까운 관측소를 계산하는 데 일회성으로 사용되며, 앱 종료 후 파기됩니다.
  • 위치 권한을 허용하지 않아도 검색 기능을 통해 앱의 모든 기능을 사용할 수 있습니다.

4. 제3자 서비스

물때는 아래 외부 서비스를 활용합니다. 개인정보는 전달되지 않습니다.

  • 국립해양조사원 (KHOA) 공공데이터 API: 조석 예보 데이터 조회. 위치·개인정보는 전달되지 않으며, 관측소 코드·날짜 등 공개 파라미터만 사용됩니다.
  • Google Fonts: 앱 폰트(Jua, Noto Sans KR) 로드. 폰트 로드 시 Google 서버에 요청이 발생할 수 있습니다.

5. 데이터 보관 및 삭제

  • 즐겨찾기·최근 관측소 데이터는 기기에만 저장되며, 앱 삭제 시 함께 삭제됩니다.
  • 앱 설정 → 데이터 초기화를 통해 언제든지 삭제할 수 있습니다.

6. 아동 개인정보 보호

물때는 만 14세 미만 아동을 대상으로 개인정보를 수집하지 않습니다. 아동의 개인정보가 수집된 사실을 인지한 경우 즉시 삭제합니다.

7. 개인정보처리방침 변경

본 방침이 변경될 경우 이 페이지를 통해 공지하며, 중요한 변경 사항은 앱 업데이트 노트에도 안내합니다.
