tag 사용방법

1. 태그는 꺽은 괄호<>를 사용하여 태그를 표시 합니다.
2. 기본적으로 태그는 여는 태그와 닫는 태그로 이루어 집니다. 물론 모두 그런것은 아닙니다.
3. 태그 안에는 속성을 넣을 수 있습니다.
4. 꺽은 괄호와 태그 사이에는 빈칸을 사용하지 않습니다.
ex) < 태그 > < 태그> <태그 >
5. 태그와 속성 사이에는 한칸을 띄워줍니다.
6. 태그 안에 속성은 여러개가 들어갈 수 있습니다. 속성과 속성 사이도 한칸을 띄워줍니다.
7. 태그와 태그는 걸쳐서 사용하면 안됩니다. ex)

html 특수문자

html 태그에서 사용하는 기호인<> 및 특수문자를 활용하기 위한 &기호 등은 html 특수문자표를 활용해야 합니다. 띄어쓰기도 html 내에서는 한칸만 인식이     됩니다.
< : &lt;
> : &gt;
& : &amp;

html 문서의 주석 사용하기

주석은 브라우저에서 해석해서 표현하지 않습니다.
퍼블리셔(개발자)가 필요한 부분에 설명글을 달거나 특정부분을 표시할 때 사용합니다.
사용법은 <!-- 한칸 띄고 주석 내용을 적어 줍니다 -->
vs code 편집기에서는 단축키 ctrl + / 또는 alt + shift + a를 사용하여 편하게 적용할 수 있습니다
