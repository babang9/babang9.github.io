html 태그의 분류

블록 속성의 태그

시맨틱 태그

header 태그 : 머리말 영역으로 문서 또는 섹션의 헤더를 나타낸다.
nav 태그 : 네비게이션(navigation)의 약어이며, 길안내를 하는 나침반과 같은 역할을 수행함. 즉 다른 문서 사이를 탐색할 수 있는 링크의 집합이다.
main 태그 : 문서의 주요 콘텐츠를 나타낸다. main 태그는 한 페이지에 한번만 사용할 수 있다.
section 태그 : 섹션이란 어느 한 구역을 말한다. 서로 관계가 있는 문서를 분리하는 역할을 한다.
article 태그 : 섹션과 사용하는 목적은 동일하며 내용이 각기 독립적이고 홀로 설 수 있는 내용을 담는다.
aside 태그 : 본문 영역의 옆에 사용하는 태그이며, 웹페이지의 사이드에 장바구니나 배너가 들어있는 상자를 aside태그로 만든다.
footer 태그 : 웹페이지의 하단 영역을 만들 때 사용하며, 회사명, 정책, 전화번호, 주소, copyright 등이 위치한다.
이 외에도 시맨틱 태그는 더 존재함

시맨틱 외의 태그

div 태그 : 의미가 없는 단순한 상자를 만들어 주는 태그이며, 레이아웃을 만들때 아직까지는 주로 사용된다.

hn 태그 : 제목을 표시하는 태그. h1 ~ h6까지 사용되며, 숫자가 적을 수록 큰 제목이다.

p 태그 : 문단을 만들 때 쓰이는 태그. 문단과 문단 간격이 생김.

ul 태그 : 순서가 없는 목록을 만드는 태그

ol 태그 : 순서가 있는 목록을 만드는 태그

dl 태그 : 정의식(사전식) 목록을 만드는 태그

이 외 다수의 블록 태그가 존재한다.

인라인 속성 태그

인라인 속성

strong 태그 : 글을 강조할 때 사용합니다.

b 태그 : strong 태그와 동일하게 글자를 두껍게 만들지만, 의미상 내용을 강조하지는 않습니다.

em 태그 : 글자를 기울어지게 만들며 내용을 강조해 줍니다.

i 태그 : em 태그와 동일하게 글자를 기울이지게 만들지만, 의미상 내용을 강조하지는 않습니다.

a 태그 : 하이퍼 링크를 만들 때 사용합니다. 네이버(_blank), 다음(_parent), 구글(_self), 페이스북(_top)

span 태그 : div와 같은 역할을 하는 인라인 속성의 태그입니다. 줄바꿈 없음
