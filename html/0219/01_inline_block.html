인라인 속성 태그와 블록 속성 태그 비교

block 속성 태그의 특징

  • block 속성의 태그는 한줄을 모두 사용합니다
  • css로 넓이와 높이 값을 적용할 수 있습니다
  • 패딩과 마진을 상하좌우 모두 적용할 수 있습니다
  • block 속성의 태그는 정해져 있습니다
  • div, p, h1~h6, ul, li, ol, dl, dt, dd 및 모든 시맨틱 태그 등이 있습니다

inline 속성 태그의 특징

  • inline 속성 태그는 태그로 감싸도 줄바꿈이 되지 않습니다
  • css style에 넓이와 높이를 넣어도 적용되지 않습니다
  • 패딩과 마진은 좌우는 적용되나 상하는 적용되지 않습니다
  • a, b, em, strong, i, span 태그 등이 있습니다

inline-block 속성 태그의 특징

  • inline-block 속성의 태그는 inline과 block 속성의 특징을 조금씩 가지고 있습니다
  • 넓이와 높이는 설정할 수 있지만, 태그 사용시 줄바꿈이 되지는 않습니다
  • img, input 태그 등이 있습니다
  • img, input 태그(inline-block적용) 등이 있습니다
