table 만들기

  • table 태그 안에는 줄을 만드는 tr 태그와 칸을 만드는 td(th) 태그를 반드시 사용해야 합니다.
  • 칸을 만드는 태그 중 th 태그는 제목칸을 만들 때 사용합니다.
  • 칸을 만드는 태그 중 td 태그는 일반칸을 만들 때 사용합니다.
  • colspan은 좌우칸을 합칠 때 사용하는 속성입니다. 값으로는 합치려는 칸의 갯수를 적습니다.
  • rowspan은 상하칸을 합칠 때 사용하는 속성입니다. 값으로는 합치려는 칸의 갯수를 적습니다.
  • 칸을 합치는 속성은 외쪽 위칸을 기준으로 적용합니다.
